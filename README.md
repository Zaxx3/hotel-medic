# Hotel Medic

The official site for hotel "Medic" located in Bulgaria, created with focus on clear information delivery, simple and minimalistic design and easy navigation for users.

## Technologies

- React
- SCSS
- Vite

## Link to the site

You can check out the site [here](https://hotel-medic.com/bul/)
_(Link will be updated once the site officially goes live.)_

## Final notes

This is my second commissioned project. While there is still room for improvement both aesthetically and functionally, the website was completed within a tight timeframe. Further revisions and refinements may be made in the future.
